# OP Gap (Operator Gap)

Author: Erik Michálek  
Project: Hidden Self  
Year: 2026

OP Gap (Operator Gap) describes the micro-interval between stimulus and reaction where response selection becomes possible.

The concept focuses on reaction awareness, attention mechanics, and trained response patterns.

This repository serves as a timestamped origin reference for the framework.
